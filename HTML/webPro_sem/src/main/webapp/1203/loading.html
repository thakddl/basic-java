JavaScript 코드는 웹 브라우저가 모든 문서의 요소들을 로드한 뒤 실행되어야 오류가 발생하지 않는다
	오류 발생 원인 : 아직 생성되기 전 html요소에 속성을 추가하려고 할 때 발생
	
	해결 방법 1,2 : window객체의 load 사용
		- 문서의 모든 콘텐츠(images, script, css, etc...)가 로드된 후 발생하는 이벤트
		- 문서에 포함된 모든 요소가 로딩된 후 실행되므로 (상대적으로) 느림
	해결 방법 3 : document객체의 DOMContentLoded 사용
		- html과 script가 로딩된 시점에서 발생하는 이벤트
		- load이벤트보다 먼저 발생되므로 실행이 더 빠르다
